# Telerivet Demo

## Description

Project using [Televiret GateWay](https://telerivet.com/) to send SMS on web browser

## Run Source

- Run API

  1. yarn install
  2. yarn start

- Run UI Test
  1. cd `test` directory
  2. open `index.html` file
